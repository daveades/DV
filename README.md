# ElleConnectel Afrique Dashboard

This workspace contains a lightweight dashboard to visualize youth unemployment patterns across African regions using your meeting writeup data.

## What is included

- Regional average youth unemployment chart
- Country-level breakdown chart with region filter
- Southern Africa youth vs adult unemployment comparison chart
- Summary metric cards and data notes

## Project structure

- `index.html`
- `src/data/africaData.js`
- `src/js/app.js`
- `src/styles/main.css`

## Run locally

Open `index.html` directly in VS Code Live Preview/Live Server or use a local static server.

PowerShell option:

```powershell
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Data note

Some values in your writeup were expressed as ranges or estimates. Midpoint values are used in the charts where needed.
